核酸配列の研究をするとき、DNA分子(GACTを使う)やRNA分子(GACU)の中のヌクレオチドの順序を示すために連続した文字を使います。 使用可能な文字はA, C, G, Tで、リン酸ジエステルの主鎖に共有結合しているDNA鎖の4つのヌクレオチド塩基 - アデニン, シトシン、グアニン、チミン - を表現します。

このミッションで、あなたは長さによって列を複数のセクションに分割し、反転の個数(number of inversions)でそれらをソートし、再び1つの列に結合する(join)必要があります。

文字"A", "C", "G", "T"の列と、部分列の長さが与えられます。 最初に与えられた列を、与えられた長さで分割する(split)必要があります。 最後の部分が短いときは、捨てます。 次にこれらの部分列を反転の個数でソートします。 反転の個数は、お互いに関してアルファベット順でないエントリーのペアの個数です。 例えば、"TAACGC"の反転の個数は6です。 内訳はTと他の文字については5で、最後の"GC"の所で1となります。 いくつかの部分列が同じ反転の個数をもっていた場合、それらは元々の列の順序にします。 最後にこれらの部分列を1つの列に結合します。 最終的な列は元々の列より短くなる可能性があります。

例を見てみましょう。 与えられた列は"ACGGCATAACCCTCGA"で分割の長さは3です。 最初は分割します:"ACG", "GCA", "TAA", "CCC", "TCG"と"A"です。 最後の"A"は捨てます。 それから反転の個数を数えます。 "ACG" - 0, "GCA" - 3, "TAA" - 2, "CCC" - 0, "TCG" - 2です。 これらをソートします: "ACG", "CCC", "TAA", "TCG", "GCA"になります。 そして結合します: "ACGCCCTAATCGGCA"。

このチャレンジにはもうひとつルールがあります。それはコードゴルフミッションだということです。つまり、できるかぎりあなたのコードを短くしてください。 短ければ短いほど点数が高くなります。このミッションのスコアは動的でコードの長さに直接関連しています。 参考まで、点数は使用した文字の個数に基づいています。200文字が許される最大で、もらえるのは0点になります。 200文字より少ない各文字について、1点もらえます。たとえばコードが150文字だったときは50点になります。 このミッションでは、空文字も数えます。ただしインデントは数えません。

入力: 2つのパラメータ。列、文字列。分割の長さ、整数。

出力: ソートされた列、文字列。
